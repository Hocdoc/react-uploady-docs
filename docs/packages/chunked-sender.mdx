## Installation

```bash npm2yarn
npm install @rpldy/chunked-sender
```

## Details

Adds chunked upload capabilities on top of the regular XHR [@rpldy/sender](../rpldy-sender)
Exposes an UploaderEnhancer that replaces the default send method the [uploader](../../api/uploader) uses.

For usage with your React app, see [@rpldy/chunked-uploady](../rpldy-chunked-uploady).

Chunked uploading doesn't support grouped uploads (in single XHR request) or URL uploading.
These will be handed over to the default [@rpldy/sender](../rpldy-sender)
