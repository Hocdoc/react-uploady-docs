## Package

**_[@rpldy/uploady](../../../packages/rpldy-uploady)_**

## Installation

```bash npm2yarn
npm install @rpldy/uploady
```

## Details

```typescript
type useAbortAll = () => () => boolean;
```

Returns abort all method.
This is a convenience method to use instead of using the Context API directly.

## Example

```javascript
import { useAbortAll } from "@rpldy/uploady";

const MyComponent = () => {
    const abortAll = useAbortAll();

    return <button onClick={() => abortAll()}>Abort All</button>
};
```
