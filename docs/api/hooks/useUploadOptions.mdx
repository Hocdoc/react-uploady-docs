## Package

**_@rpldy/uploady_**

## Installation

```bash npm2yarn
npm install @rpldy/uploady
```

## Details

```typescript
type useUploadOptions = (options?: CreateOptions) => CreateOptions;
```

- See: [CreateOptions](../types#createoptions)

Hook to set/get upload options.

## Example

```javascript
import { useUploadOptions } from "@rpldy/uploady";

const MyComponent = () => {
    const options = useUploadOptions({ grouped: true, maxGroupSize: 3 });

    //...
};
```
